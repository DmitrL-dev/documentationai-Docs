## Overview

GitHub empowers you to build, scale, and deliver secure software with AI-powered tools. As the world's largest developer platform, it hosts over 100 million users and serves 90 of the Fortune 100 companies. Founded in 2008 and acquired by Microsoft in 2018, GitHub revolutionized code collaboration through version control, continuous integration, and now advanced AI features like GitHub Copilot.

<Callout kind="info">
GitHub combines Git's power with a vast ecosystem, enabling you to collaborate seamlessly while AI accelerates your workflow.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create a free account at [github.com/signup](https://github.com/signup).
  </Step>
  <Step title="Create Repository" icon="repo">
    Click the `+` icon and select "New repository". Name it `my-first-repo`.
  </Step>
  <Step title="Clone and Push" icon="git-branch">
    Use GitHub CLI or standard Git commands.

    <CodeGroup tabs="CLI,git">
      ```bash
      gh repo clone YOUR_USERNAME/my-first-repo
      echo "# My First Repo" > README.md
      git add .
      git commit -m "Initial commit"
      git push origin main
      ```
      ```bash
      git clone https://github.com/YOUR_USERNAME/my-first-repo.git
      cd my-first-repo
      echo "# My First Repo" > README.md
      git add .
      git commit -m "Initial commit"
      git push origin main
      ```
    </CodeGroup>
  </Step>
</Steps>
