<Update label="2024-10-15" description="v1.2.0" tags={["feature", "security"]}>
## New Features

- Introduced AI-powered code review suggestions using GitHub Copilot, helping you catch bugs before they merge.
- Added real-time collaboration in pull requests with live cursors and inline comments.

## Security Updates

- Enhanced repository scanning with secret detection for over 100 new token patterns.
- Implemented zero-trust access controls for Actions workflows.

## Improvements

- Reduced pull request merge times by 40% with optimized diff rendering.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>
## New Features

- Launched GitHub Models playground for testing AI models directly in your browser.
- New collaboration tool: @mentions in wikis now support threaded discussions.

## Bug Fixes

- Fixed intermittent failures in Codespaces startup, now succeeding 99.9% of the time.
- Resolved API rate limiting issues for Dependabot alerts.

<CodeGroup tabs="Before,After">
````javascript
// Before: Manual dependency check
const deps = require('./package.json');
if (deps.dependencies['lodash'] < '4.17.21') {
  console.warn('Update lodash');
}
````
````javascript
// After: Using GitHub API for automated alerts
const response = await fetch('https://api.github.com/repos/user/repo/dependabot/alerts');
const alerts = await response.json();
alerts.forEach(alert => console.log(`Update ${alert.package.name}`));
````
</CodeGroup>

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>
## Breaking Changes

- Deprecated legacy GraphQL v4 endpoints; migrate to v5 using the new schema explorer.

## New Features

- Integrated GitHub Copilot Chat in the VS Code extension for natural language code generation.

## Deprecations

- Removed support for SAML 1.1; upgrade to SAML 2.0 for enterprise SSO.

</Update>
